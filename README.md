# Предметная область
Система учитывает всё установленное программное обеспечение в организации, компьютеры на которых это ПО установлено. Благодаря этому можно оптимизировать лицензионные расходы и лучше распределять бюджет.
Например, приезжает новая партия компьютеров, работник отвечающий за отдел программного обеспечения\сотрудник «привязанный» к этому компьютеру регистрирует его в систему, система сканирует файлы и автоматически проверяет наличие необходимого ПО. При любом раскладе данные отправляются в систему вместе с подсчтами на лицензии. Эти данные могут просматривать работники отвечающее за отдел программного обеспечения.
Отчет включает в себя такие данные как:
- Имя компьютера, серийный номер
- Сотрудник, закрепленный за этим компьютером
- Информацию о том, какое программное обеспечение имеется на компьютере, а какое отсутствует
- Сумму в денежном эквиваленте за каждый отдельный продукт и в совокупности
# Цели системы: какие задачи она решает.
- Автоматизирует учет программного обеспечения
- Имеет удобный доступ к большой безе данных, в которой хранятся данные об отсутствии/присутствии программного обеспечения на каждом компьютере   
- Управляет затратами на ПО

# Определить пользователей системы (актеров) и их роли.
- Работники отвечающие за отдел программного обеспечения
- Сотрудники компании

# Составить список основных функций, которые система должна выполнять.
- Авторизация компьютеров в системе
- Проверка наличия ПО на компьютере, подсчет цены на ПО, создание отчета

- Просмотр данных в системе
